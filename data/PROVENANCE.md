## Generation procedure

Both files were generated by `src/generate_scenarios.py` calling the
Anthropic API:

| Parameter | Value |
|---|---|
| Generation model | `claude-opus-4-7` (set in `src/config.py:GEN_MODEL`) |
| `max_tokens` | 2000 |
| `temperature` | API default (1.0) — *not* explicitly set in the request |
| Seed | none — Anthropic's API does not support seeded generation |
| Batch size | 5 scenarios per API call (chosen to fit token budgets reliably) |
| Domains | 6 domains × 25 scenarios = 150 (see `src/config.py:DOMAINS`) |
| Pilot domains | 6 domains × ~3–4 scenarios = 20 (see `src/config.py:PILOT_DOMAIN_COUNTS`) |

The generation prompt instructs Claude Opus to stack physical fear-sensations
(*hands shaking, heart hammering, throat tight, pulse in ears, knees of water,
vision narrowing, mouth dry*) at least three deep per option, and to frame
Option B as *body-relief* rather than moral self-justification. The full prompt
template is `GEN_PROMPT` in `src/generate_scenarios.py`. See paper §3.4 for the
methodological rationale.

## Reproducibility note

Anthropic's API at the time of generation did not expose seeding controls and
the request used default sampling. **Re-running `generate_scenarios.py` will
produce a different 150-scenario set**, even with the same model id and
prompt. The bundled `scenarios.csv` here is the *exact* set used for every
result in the paper; readers reproducing paper numbers should use it as-is
rather than regenerating.

If a future reader wants to test the *robustness* of the headline result to
scenario sampling variability, they can use `generate_scenarios.py` to produce
a fresh 150-scenario set and rerun the eval pipeline; the paper makes no claim
that the specific 150 in `scenarios.csv` are uniquely necessary. The pilot gate
(§3.3) would need to be re-passed for the new set to qualify as a comparable
fear-axis evaluation.

## Pilot history

`pilot.csv` is the *visceral-fear-prompt* pilot (v2). An earlier
*deliberative-prose* pilot (v1) failed the gate (mean afraid +0.045, mean
moral-judgment-arousal +0.044 — near-equal, indicating both axes engaged).
The v2 prompt was rewritten to stack physical fear-sensations and yielded
mean afraid +0.075, mean moral-judgment-arousal −0.005 — a substantially
cleaner profile that passed the gate. v1 is not bundled. See
`results/pilot_validation.json` for the v2 numbers.
